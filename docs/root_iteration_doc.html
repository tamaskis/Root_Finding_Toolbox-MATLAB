root_iteration

Function iteration method for finding the root of a univariate, scalar-valued function.

Syntax

x = root_iteration(f,x0)
x = root_iteration(f,x0,opts)
[x,k] = root_iteration(__)
[x,k,x_all] = root_iteration(__)

Description

x = root_iteration(f,x0) returns the root of a univariate, scalar-valued function specified by the function handle f, where x0 () is an initial guess of the root.

x = root_iteration(f,x0,opts) does the same as the syntax above, but allows for the specification of optional solver parameters. opts is a structure with fields detailed in the "Input/Output Parameters" section below.

[x,k] = root_iteration(...) also returns the number of iterations (k) performed of the secant method.

[x,k,x_all] = root_iteration(...) does the same as the previous syntaxes, but also returns an array (x_all) storing the root estimates at each iteration. This syntax requires that opts.return_all be set to true.

Example #1: Root of a simple function.

Find the root of that is in the interval .

Defining ,

f = @(x) x^2-1;

We want a root in the interval . We solve for this root using the function iteration method with an initial guess of .

x = root_iteration(f,1.9)

x =

    1.0475

While we obtain a result close to the true result of 1, the function iteration method never converges to within the default tolerance (TOL = ). Additionally, this example is quite sensitive to the initial guess and can often fail to converge.

Example #2: Plot of root estimates at each iteration.

In Example #1, we found the positive root of using the secant method. Now, produce a plot of all root estimates obtained by the root_iteration function during its solution procedure.

First, we define in MATLAB like before.

f = @(x) x^2-1;

To obtain the root estimates at all iterations, we simply need to define the opts structure, set its return_all field to true, and pass it to the root_iteration function. (Note: we use the same initial guess as in Example #1).

opts.return_all = true;
[~,~,x_all] = root_iteration(f,1.9,opts);

Plotting the root estimates at each iteration,

figure;
plot(x_all,'k*','MarkerSize',9,'LineWidth',1.5);
grid on;
xlabel('Iteration','Interpreter','latex','FontSize',18);
ylabel('Root Estimate','Interpreter','latex','FontSize',18);
